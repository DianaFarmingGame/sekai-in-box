# Sekai In Box

于Godot之上构建的，基于Lisp的高灵活性二维游戏引擎。

NOTE: 请使用以下命令克隆仓库，以防止历史的动态链接库占用太多存储：

```shell
git clone --depth 1 https://github.com/DianaFarmingGame/sekai-in-box.git
```

如要使用 SSH，请使用以下命令：

```shell
git clone --depth 1 git@github.com:DianaFarmingGame/sekai-in-box.git
```
